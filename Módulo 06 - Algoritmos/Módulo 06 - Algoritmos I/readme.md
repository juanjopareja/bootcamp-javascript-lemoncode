## Descripción  
Carrito de compra dinámico donde el usuario pueda elegir las unidades que desea de cada producto, a partir de una lista de productos dada.
Tendremos una sección describiendo la lista de productos y selectores para elegir las unidades deseadas, un botón para calcular la factura, y una zona con la factura desglosada mostrando el subtotal, impuestos y el importe total.
